HTML Encoder / Decoder

Escape text to HTML entities so it can be safely embedded in markup, or decode HTML entities back into plain text — supports both numeric and named entities.

Encode converts < > & " ' to named and numeric entities; Decode restores them.

How it works

  1. Paste your text into the Input area.
  2. Click Encode to convert & < > " and ' to safe HTML entities, or Decode to reverse the process.
  3. Decode recognizes the 5 XML entities plus common named entities such as &nbsp;, &copy;, &euro;, &pound;, &yen;, &hellip; and numeric entities like &#65; and &#x41;.
