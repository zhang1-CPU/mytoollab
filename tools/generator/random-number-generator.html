Random Number Generator

Generate random integers or decimal numbers in a range. Toggle uniqueness, sort results, then copy to clipboard or download as a text file.

How it works

  1. Set a range using minimum and maximum values (they can be negative or decimals).
  2. Choose how many numbers to generate and whether you want integers or decimals.
  3. Toggle uniqueness and sorting, then copy to clipboard or download as a text file.

What is a Random Number Generator?

A random number generator produces unpredictable numeric values within a specified range using cryptographically secure randomness — suitable for games, statistical sampling, and security applications.

How to use

  1. Step 1: Enter the minimum and maximum range values.
  2. Step 2: Choose a count (1-50) and the type of number (integer or decimal).
  3. Step 3: Toggle uniqueness and sort options if desired.
  4. Step 4: Click 'Generate'. Copy the results or download as a .txt file.

Use cases

  • Selecting random winners for giveaways and contests.
  • Generating random samples for statistics and research.
  • Rolling dice and shuffling in online games.
  • Creating random test data for software development.

Frequently Asked Questions

How random are the numbers?

The generator uses cryptographically secure random number generation (crypto.getRandomValues), suitable for security-sensitive applications.

Can I generate multiple numbers at once?

Yes. Enter the quantity you want (up to 50) and all numbers will be generated at once.

What happens if I choose 'unique' but my range is too small?

If the range doesn't have enough unique integers, the tool warns and switches to allowing duplicates automatically.
