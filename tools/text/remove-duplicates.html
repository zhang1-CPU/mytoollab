How it works

  1. Paste lines into the input box — one entry per line.

Frequently Asked Questions

How does this tool find duplicates?

The tool compares each line with others to find exact matches. Enable 'Ignore case' to treat 'Apple' and 'APPLE' as duplicates, or 'Trim spaces' to ignore leading/trailing whitespace.

What does 'Keep First' vs 'Keep Last' mean?

'Keep First' keeps the first occurrence and removes subsequent duplicates. 'Keep Last' keeps the last occurrence and removes earlier ones.

Can I remove blank lines?

Yes. Enable 'Remove blank lines' to automatically remove empty lines from your text.

Does this work with sorted lists?

Yes. This tool removes exact duplicate lines regardless of order. If you want sorted unique lines, copy the output to our Case Converter or another text tool.

How does case sensitivity work?

When 'Ignore case' is enabled, all lines are compared in lowercase. So 'Hello', 'HELLO', and 'hello' would be treated as duplicates.
