How it works

  1. Open this page — your device provides the browser details instantly.
  2. Your public IP is resolved via a public API (see the data sources).
  3. Click Copy JSON or Download JSON to take the snapshot elsewhere.

What is it?

A client-side snapshot tool — it reads the browser's own data (User-Agent, screen, time zone, language, color depth, device pixel ratio, Do Not Track) and adds your public IP from a JSON endpoint. No account, no tracking, no cookies set by this page.

Use cases

  • Quickly sharing your IP or User-Agent with support teams during debugging.
  • Verifying what headers and resolution a browser reports from a specific device.
  • Including a device snapshot alongside bug reports.

FAQs

Why is the IP not detected?

The IP lookup requires a network call to a public JSON service. If that call is blocked by a firewall or ad blocker, the IP section will show "Not available". Try refreshing or disabling the blocker temporarily.

Is anything stored on zlbox servers?

No. The page renders client-side; IP resolution is done via a third-party JSON endpoint. The page does not record anything.
