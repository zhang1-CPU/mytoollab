URL Encode / Decode

Convert plain text to percent-encoded URL-safe format, or decode URL-encoded strings back to readable text.

URL encoding converts special characters (spaces, punctuation, non-ASCII text) into percent-encoded format like %20 (space) or %C3%A9 (é). It is required for URL parameters and safe HTTP transmission.
