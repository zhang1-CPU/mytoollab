How it works

  1. Enter an integer and choose its source base. Leading 0x, 0b, and 0o are stripped automatically, and negative signs are preserved.
  2. All calculations use BigInt, so arbitrarily large integers convert exactly — no overflow, no precision loss.
  3. Click any result to copy it to your clipboard. Digits above 9 use the alphabet a–z (case-insensitive on input).

What is a Number Base Converter?

A base converter translates integers between positional numeral systems — binary (base 2), octal (base 8), decimal (base 10), hexadecimal (base 16), or any base up to 36. Programmers, students, and engineers use it daily.

Use cases

  • Debugging memory addresses and bit flags in hex or binary.
  • Converting color codes between decimal, hex, and RGB.
  • Studying number theory and discrete mathematics.
  • Converting IDs and serial numbers in software engineering.

Frequently Asked Questions

Does it support very large numbers?

Yes. Conversion uses BigInt so integers can be thousands of digits long with no rounding.

Are negative numbers supported?

Yes. Prefix your input with a minus sign (e.g. -128). Negative values are preserved in every output base.
