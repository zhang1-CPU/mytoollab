Remove Duplicate Lines

Paste text and instantly remove duplicate lines. Toggle ignore case, trim whitespace, blank-line removal, and sort order — live counts and copy/download included.

Ready.

How it works

  1. Paste text into the input area, one item per line. Changes update the output instantly.
  2. Toggle ignore case, trim whitespace, remove blank lines, and pick a sort order.
  3. Click Copy or Download to take the cleaned list elsewhere.

What is it?

A line-level dedupe tool: given a list with repeats, it produces a clean, unique list according to your preferences.

Use cases

  • Cleaning keyword lists, email lists, and product SKUs.
  • Preparing CSV/TSV files by removing duplicate rows.
  • Deduplicating scraped data before analysis.
  • Normalizing user-entered tags and categories.

Frequently Asked Questions

Is the original order preserved?

Yes, unless you select sorting. The first (or last) occurrence is kept in its original position.

How does ignore-case work?

"Apple" and "apple" are treated as duplicates. The first (or last) occurrence in the source is kept, preserving the original character case.
