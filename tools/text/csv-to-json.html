CSV ↔ JSON Converter

Convert between CSV and JSON instantly. Handles quoted fields with commas and newlines, auto-detects delimiters, and supports pretty-formatted output.

Auto-detects delimiter (comma, tab, or semicolon). Supports quoted fields with embedded commas, quotes, and newlines.

How it works

  1. Paste CSV or JSON into the Input area. The converter auto-detects the delimiter and type.
  2. For CSV, check First row is header to produce an array of objects; otherwise the output is a 2D array.
  3. Quoted fields with embedded commas, double-quotes, and newlines are supported.
