What is Case Converter?

A case converter transforms text between different letter casing conventions: UPPERCASE, lowercase, Title Case, Sentence case, camelCase, PascalCase, snake_case, kebab-case, and more. It's essential for developers and writers working with code and documents.

How to use

  1. Step 1: Paste or type your text into the input area.
  2. Step 2: Click any case style button to convert instantly.
  3. Step 3: Copy the result with one click.
  4. Step 4: The original text is preserved for trying different styles.

Use cases

  • Converting variable names for programming (camelCase, snake_case).
  • Formatting titles and headings for documents.
  • Converting messy copied text to proper formatting.
  • Preparing text for URLs, filenames, or identifiers.

What case types are supported?

Uppercase, lowercase, Title Case, Sentence case, camelCase, PascalCase, snake_case, kebab-case, constant case, dot.case, and path/case.

Yes. Numbers, symbols, and punctuation are preserved exactly as entered.
