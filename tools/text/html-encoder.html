HTML Encoder / Decoder

Escape text to HTML entities for safe embedding, or decode entities back to plain text. Supports &amp; &lt; &gt; &quot; &#39; &nbsp; plus numeric decimal and hex references.

How it works

  1. Encode panel: Type plain text on the left. Special characters become safe HTML entities in real time.
  2. Decode panel: Paste HTML with entities on the right. Named (e.g. &copy;), decimal (&#65;) and hex (&#x41;) references all get decoded.
  3. Copy / Download: One-click clipboard copy or download as .txt file — no uploads, everything is local.

What is HTML Encoder / Decoder?

HTML encoding converts reserved characters like <, >, &, ", and ' into safe entity references that browsers will display as text rather than interpret as markup. Decoding reverses the transformation, useful when extracting or auditing HTML content.

Use cases

  • Embedding code snippets in blog posts and documentation.
  • Preparing user-generated content to prevent XSS rendering issues.
  • Extracting text from HTML sources and translating entities to Unicode.

Frequently Asked Questions

What is the difference between &#160; and a normal space?

&#160; (or &nbsp;) is a non-breaking space. Visually identical but HTML will not wrap the line at that position.

Do numeric references support any Unicode code point?

Yes. Decimal (&#DDD;) and hex (&#xHHHH;) references can represent any Unicode code point.
