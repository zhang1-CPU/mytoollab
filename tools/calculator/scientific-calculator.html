How it works

  1. Click or tap keys to build your expression; trig functions expect degrees (e.g., sin(30) = 0.5).
  2. Use parentheses ( ) to control the order of evaluation and compose nested math.
  3. Press = or hit Enter to compute. Use C to clear or ⌫ to delete the last character.

Frequently Asked Questions

Do trigonometric functions use degrees or radians?

This calculator uses degrees for all trig functions (sin, cos, tan). For example, sin(30) = 0.5 and cos(60) = 0.5.

What functions are available?

Available functions include: sin, cos, tan (trigonometry), log (base-10 logarithm), ln (natural logarithm), sqrt (square root), x^2 (square), x^y (power), n! (factorial), and parentheses for grouping.

How do I use the power function?

Click x^y, then enter the exponent. For example, to calculate 2^8, press: 2, x^y, 8, =. The result should be 256.

Can I use keyboard shortcuts?

Yes. Use ^ for power, % for percentage, and standard keyboard operators (+, -, *, /). Enter evaluates, Backspace deletes, Escape clears.
