IP Address Generator

Generate random IPv4 or IPv6 addresses in bulk. Pick public networks, private ranges (10.x, 172.16-31.x, 192.168.x), or supply your own CIDR. Copy or download your list in one click.

How it works

  1. Pick IPv4 or IPv6 and how many addresses you want (1–50).
  2. Choose a preset (public / private / common CIDRs) or supply a custom CIDR like 10.1.0.0/16.
  3. All addresses are generated locally in your browser using cryptographically strong randomness.

What is this tool?

An IP Address Generator produces random IP addresses for testing firewall rules, load balancer configs, QA test data, or documentation examples. All generation happens entirely in your browser.

How to use

  1. Step 1: Choose IP version (IPv4 or IPv6).
  2. Step 2: Set a count (1–50).
  3. Step 3: Pick a range preset, or choose 'Custom CIDR' and enter a range.
  4. Step 4: Copy the whole list or download as a text file.

Use cases

  • Generating example addresses for documentation and diagrams.
  • Populating test data for network tooling / monitoring dashboards.
  • Testing ACL rules, firewall configs, and IP allow/deny lists.
  • Load testing scripts that need realistic source addresses.

Frequently Asked Questions

Are these real, routable IPs?

Generated addresses follow valid syntax, but they may or may not correspond to actual hosts on the internet. Private-range IPs are not globally routable by design.

What's a CIDR?

CIDR notation (e.g. 192.168.1.0/24) describes a network and its prefix length. /24 means the first 24 bits are the network, leaving 8 bits (256 addresses) for hosts.

Can I generate IPv6 in a specific subnet?

IPv6 support is limited to fully random addresses in this release. Custom IPv6 CIDRs are coming soon.
