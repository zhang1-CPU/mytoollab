What is PDF metadata?

PDF metadata is a small set of properties embedded inside a PDF file. It typically includes the document title, author, subject, keywords, creation and modification dates, the producing application, and the PDF version.

How to use

  1. Step 1: Click 'PDF File' and choose a .pdf file from your device.
  2. Step 2: Wait a moment while the file is parsed locally.
  3. Step 3: Review the extracted information.

About PDF Info Viewer

Our free PDF info viewer extracts and displays comprehensive metadata from any PDF document, giving you instant access to details that are normally hidden deep inside file properties. From basic information like page count and file size to technical details like PDF version, producer software, creation and modification dates, and embedded metadata fields, this tool reveals everything about a PDF without requiring Adobe Acrobat or any other specialized software.

The tool works entirely in your browser — simply upload or drag-and-drop a PDF to instantly scan its structure and display a clean, organized summary of all available information. Whether you are auditing documents for compliance, verifying the source of a received file, checking document history, or just curious about a PDF's origin, this utility provides answers in seconds.

Frequently Asked Questions

What metadata can this tool extract from a PDF?

The tool extracts all standard PDF metadata fields including the document title, author, subject, keywords, creator application (which software created the PDF), producer (which software processed or converted it), creation date, modification date, and PDF version number.

Is my PDF uploaded to a server?

No. All processing is performed locally in your web browser using JavaScript. Your PDF file never leaves your device and is not stored anywhere.

Can I view the actual text content of the PDF?

No. This tool only extracts metadata and structural information. To read the text content, use our PDF to Text converter instead.

What if some fields are empty or not shown?

Not all PDFs contain every metadata field. Some fields may be empty or not present if the PDF was created without specifying them. The tool only displays fields that actually exist in the file.

Can I edit the PDF metadata with this tool?

No. This is a read-only viewer that displays existing metadata. To edit metadata, you would need a PDF editor with metadata modification capabilities.
