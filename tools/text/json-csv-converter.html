Frequently Asked Questions

What is nested JSON flattening?

Nested JSON like {"user":{"name":"John"}} can be flattened to user.name="John" using dot notation. Enable the 'Flatten nested objects' option to automatically convert nested structures.

How does array expansion work?

When an object contains arrays, enabling 'Expand arrays as separate rows' creates one row per array item. For example, {"name":"John","items":["A","B"]} becomes two rows.

What CSV formats can I convert?

You can auto-detect delimiters or manually choose comma, semicolon, or tab. The converter handles quoted fields and escaped quotes correctly.

How do I convert CSV to JSON?

Click the 'CSV to JSON' tab, paste your CSV, choose delimiter settings, and click 'Convert to JSON'. Use 'First row as keys' if your CSV has headers.

Can I download the converted file?

Yes. Use the 'Download CSV' or 'Download JSON' buttons to save the converted file directly to your device.
