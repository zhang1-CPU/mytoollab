Compress PDF

Reduce PDF file size by stripping redundant bytes and metadata. See original vs. optimized size instantly and download the result. Everything runs in your browser — nothing is uploaded.

Drop a PDF here or click to choose a file
Maximum recommended: 200 MB. Works best on simple PDFs without heavy embedded images.

How it works

  1. Select or drop a PDF file on the drop zone. File is read locally in your browser.
  2. Click Compress. The tool rewrites the PDF, removing redundant bytes and optional metadata.
  3. See how much smaller the result is, then click Download to save the compressed PDF.

What is it?

A lightweight, browser-only PDF optimizer. It re-encodes the PDF to remove redundant bytes (extra whitespace, trailing nulls, and optional metadata) without re-rendering or re-encoding images, which keeps page fidelity but limits savings on image-heavy files.

Use cases

  • Reducing size of text-heavy reports, invoices, and scanned-text PDFs.
  • Stripping author/creation metadata before sharing.
  • Preparing PDFs for email attachments or web uploads.

FAQs

Will compression always reduce the file?

Many PDFs save 10-30%; however, already-optimized or image-heavy PDFs may not compress much. If no significant savings are possible, the tool reports this honestly.

Are images re-encoded?

No. This tool only strips redundant bytes and metadata. Images inside the PDF are preserved exactly.

Is my file uploaded?

No. Compression happens in your browser using JavaScript. Nothing is sent to a server.
