SQL Formatter & Minifier

Paste your SQL and instantly see a clean, indented version with uppercase keywords. Minify, copy, or download as .sql — everything runs in your browser.

How it works

  1. Paste a SQL query into the input box. Live formatting updates the output as you type.
  2. Toggle uppercase keywords, indent size, and comma handling to taste.
  3. Click Format to pretty-print, Minify to collapse whitespace, then copy or download as a .sql file.

What is it?

A browser-side SQL formatter/minifier. It uppercases common SQL keywords, breaks lines on major clauses, and indents nested parentheses — producing a readable layout without external services.

Use cases

  • Beautifying legacy SQL dumps and one-liner queries.
  • Preparing queries for code reviews and documentation.
  • Minifying queries before embedding them into scripts or config files.

Frequently Asked Questions

Which SQL dialects are supported?

Any dialect using standard clauses (SELECT, FROM, WHERE, JOIN, GROUP BY, HAVING, ORDER BY, LIMIT, INSERT, UPDATE, DELETE, WITH, CASE, UNION, INTERSECT, EXCEPT).

Is the SQL validated?

This tool focuses on formatting and whitespace. Syntax validity is not checked — please run your query in a real database engine for validation.
