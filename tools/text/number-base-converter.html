How it works

  1. Type a number and choose its current base (2 to 36). Digits use 0–9 followed by a–z for bases over 10.
  2. The input is parsed using BigInt, so integers can be arbitrarily large with no overflow.
  3. The value is simultaneously converted to binary, octal, decimal, hex, Base32, and Base36 for easy comparison.
