What is Case Converter?

A case converter transforms text between different letter-casing conventions used in programming, writing, and file naming — UPPERCASE, lowercase, Title Case, Sentence case, camelCase, PascalCase, snake_case, and kebab-case.

Use cases

  • Converting variable and function names across coding styles (camelCase, snake_case, PascalCase).
  • Standardizing headings and titles for documents and blog posts.
  • Cleaning up copied text from emails, chats, and the web.
  • Preparing filenames and URL slugs (kebab-case).

Frequently Asked Questions

What's the difference between camelCase and PascalCase?

camelCase starts with a lowercase letter (e.g., userName). PascalCase capitalizes each word including the first (e.g., UserName). JavaScript/Java favor camelCase for variables, while C# and TypeScript types use PascalCase.

When should I use snake_case vs kebab-case?

snake_case (underscores, user_id) is common in Python, Ruby, SQL column names, and PHP. kebab-case (hyphens, user-id) is used for URLs, CSS classes, and file names in multi-word file systems.

Are numbers and punctuation preserved?

Yes. Numbers, symbols, and punctuation from the original input are preserved in each case style; only letters are transformed.

Is my data sent to a server?

No. All conversion happens locally in your browser using pure JavaScript. Your text never leaves your device.
