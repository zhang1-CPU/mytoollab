Hash Generator

Compute MD5, SHA-1, SHA-256, SHA-384, and SHA-512 hashes instantly for any text or file. Results shown in hex and Base64.

Enter text or choose a file, then press Compute Hashes.

How it works

  1. Paste text into the editor, or click Choose file to hash any file (files larger than 50 MB are not recommended in the browser).
  2. Click Compute Hashes. MD5 runs in pure JavaScript; SHA variants use the browser's Web Crypto API.
  3. Each hash is shown in hexadecimal and Base64. Click Copy on any result to copy it to the clipboard.
