Frequently Asked Questions

How does the diff algorithm work?

The tool uses the Longest Common Subsequence (LCS) algorithm to find the minimum set of additions and deletions needed to transform one text into another.

What view modes are available?

There are two modes: 'Side by Side' shows both texts with differences highlighted in parallel columns. 'Inline' shows all changes in a single column with +/- prefixes.

What do the colors mean?

Green (Added) indicates new lines. Red (Removed) indicates deleted lines. Yellow (Changed) indicates modified lines (shown in inline view only).

Can I copy the diff result?

Yes. Click 'Copy Result' to copy the diff in a plain text format with +/- prefixes for easy reading.

Can I download the diff?

Yes. Click 'Download Diff' to save the comparison results as a .txt file.
