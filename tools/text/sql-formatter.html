SQL Formatter

Paste your SQL query and get a nicely indented, readable version instantly — processed entirely in your browser.

How to Format SQL

  1. Paste SQL: Copy your SQL query into the "Input SQL" box.
  2. Choose options: Toggle UPPERCASE keywords and choose indent style.
  3. Format: Click "Format SQL" — the tool restructures your query.
  4. Copy: Click "Copy" to copy the formatted SQL to your clipboard.

Frequently Asked Questions

Is this SQL formatter free?

Yes. Paste SQL and beautify it without signup.

Which SQL keywords are supported?

Common keywords like SELECT, FROM, WHERE, JOIN, GROUP BY, ORDER BY, INSERT INTO, VALUES, UPDATE, SET, DELETE are recognized.

Can I indent differently?

You can choose between 2-space, 4-space, and tab indentation.

Do you store my SQL?

No. All formatting runs locally in your browser.

Is there an uppercase option?

Yes. Keywords are uppercased automatically when the toggle is on.
